## CAGED 和弦系统

> CAGED 是吉他学习的核心系统，由 C、A、G、E、D 五个开放和弦组成

### C 和弦

> 组成音：**C(1) - E(3) - G(5)**（根音 - 大三度 - 纯五度）

```
       1品    2品    3品
1弦 ║--○---|------|------║  E  [3]
2弦 ║--●---|------|------║  C  [1] *
3弦 ║--○---|------|------║  G  [5]
4弦 ║------|--●---|------║  E  [3]
5弦 ║------|------|--●---║  C  [1] * 低音
6弦 ║--x---|------|------║
```

### A 和弦

> 组成音：**A(1) - C#(3) - E(5)**（根音 - 大三度 - 纯五度）

```
       1品    2品    3品
1弦 ║--○---|------|------║  E  [5]
2弦 ║------|--●---|------║  C# [3]
3弦 ║------|--●---|------║  A  [1] *
4弦 ║------|--●---|------║  E  [5]
5弦 ║--○---|------|------║  A  [1] * 低音
6弦 ║--x---|------|------║
```

### G 和弦

> 组成音：**G(1) - B(3) - D(5)**（根音 - 大三度 - 纯五度）

### D 和弦

> 组成音：**D(1) - F#(3) - A(5)**（根音 - 大三度 - 纯五度）

```
       1品    2品    3品
1弦 ║------|--●---|------║  F# [3]
2弦 ║------|------|--●---║  D  [1] *
3弦 ║------|--●---|------║  A  [5]

```

> **图例**：○ 空弦 | ● 按弦 | x 不弹 | * 根音 | ║ 品丝

## 音级 (Scale Degrees)

> 音级是音阶中每个音的位置编号，用于描述音与主音的关系

### 自然大调音级

| 音级 | 名称 | 功能 | 与主音关系 |
|:---:|:---:|------|:---:|
| **Ⅰ** | 主音 (Tonic) | 调性中心，最稳定 | 纯一度 |
| **Ⅱ** | 上主音 (Supertonic) | 过渡性，倾向Ⅰ或Ⅲ | 大二度 |
| **Ⅲ** | 中音 (Mediant) | 决定大/小调色彩 | 大三度 |
| **Ⅳ** | 下属音 (Subdominant) | 次稳定，推动进行 | 纯四度 |
| **Ⅴ** | 属音 (Dominant) | 强烈倾向主音 | 纯五度 |
| **Ⅵ** | 下中音 (Submediant) | 关系小调主音 | 大六度 |
| **Ⅶ** | 导音 (Leading Tone) | 强烈倾向主音（半音） | 大七度 |

### 音级与和弦

> 每个音级上可以构建三和弦，形成调内和弦

| 音级 | 和弦性质 | C大调示例 | 功能 |
|:---:|:---:|:---:|------|
| Ⅰ | 大三和弦 | C | 主功能 |
| Ⅱ | 小三和弦 | Dm | 下属功能 |
| Ⅲ | 小三和弦 | Em | 主功能 |
| Ⅳ | 大三和弦 | F | 下属功能 |
| Ⅴ | 大三和弦 | G | 属功能 |
| Ⅵ | 小三和弦 | Am | 主功能 |
| Ⅶ | 减三和弦 | Bdim | 属功能 |

### 功能和声

```
┌─────────────────────────────────────────────────┐
│                    和声功能                      │
├─────────────────────────────────────────────────┤
│  主功能 (T)    →    下属功能 (S)    →    属功能 (D)    →    主功能 (T)  │
│   Ⅰ Ⅲ Ⅵ              Ⅱ Ⅳ                 Ⅴ Ⅶ               Ⅰ        │
│  (稳定)            (推动)              (紧张)            (解决)       │
└─────────────────────────────────────────────────┘
```

## 12个调的自然大调音阶

> 自然大调音阶遵循 **全-全-半-全-全-全-半** 的音程模式

### 无升降号

| 调 | 音阶 | 记忆要点 |
|:---:|------|---------|
| **C** | C - D - E - F - G - A - B | 白键音阶，基础调 |

### 升号调 (♯)

| 调 | 升号数 | 升号位置 | 音阶 |
|:---:|:---:|------|------|
| **G** | 1♯ | F♯ | G - A - B - C - D - E - F♯ |
| **D** | 2♯ | F♯ C♯ | D - E - F♯ - G - A - B - C♯ |
| **A** | 3♯ | F♯ C♯ G♯ | A - B - C♯ - D - E - F♯ - G♯ |
| **E** | 4♯ | F♯ C♯ G♯ D♯ | E - F♯ - G♯ - A - B - C♯ - D♯ |
| **B** | 5♯ | F♯ C♯ G♯ D♯ A♯ | B - C♯ - D♯ - E - F♯ - G♯ - A♯ |
| **F♯** | 6♯ | F♯ C♯ G♯ D♯ A♯ E♯ | F♯ - G♯ - A♯ - B - C♯ - D♯ - E♯ |

### 降号调 (♭)

| 调 | 降号数 | 降号位置 | 音阶 |
|:---:|:---:|------|------|
| **F** | 1♭ | B♭ | F - G - A - B♭ - C - D - E |
| **B♭** | 2♭ | B♭ E♭ | B♭ - C - D - E♭ - F - G - A |
| **E♭** | 3♭ | B♭ E♭ A♭ | E♭ - F - G - A♭ - B♭ - C - D |
| **A♭** | 4♭ | B♭ E♭ A♭ D♭ | A♭ - B♭ - C - D♭ - E♭ - F - G |
| **D♭** | 5♭ | B♭ E♭ A♭ D♭ G♭ | D♭ - E♭ - F - G♭ - A♭ - B♭ - C |
| **G♭** | 6♭ | B♭ E♭ A♭ D♭ G♭ C♭ | G♭ - A♭ - B♭ - C♭ - D♭ - E♭ - F |

### 快速记忆法

**升号顺序**：F → C → G → D → A → E → B（口诀：**F**ather **C**harles **G**oes **D**own **A**nd **E**nds **B**attle）

**降号顺序**：B → E → A → D → G → C → F（口诀：**B**attle **E**nds **A**nd **D**own **G**oes **C**harles' **F**ather）

**找主音技巧**：
- 升号调：最后一个升号 **升高半音** = 主音（如 F♯ C♯ → C♯ 升半音 = D大调）
- 降号调：倒数第二个降号 = 主音（如 B♭ E♭ A♭ → E♭大调）
